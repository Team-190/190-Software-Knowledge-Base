# SystemCore

The SystemCore is the robot controller for FRC robots post 2026. As of right now, there is no official non-beta guide for setting up a SystemCore, but when this is released, it will go here, as well as its technical specifications document

Hover over each feature to see what it does. Click on a feature to see more relevant information!
